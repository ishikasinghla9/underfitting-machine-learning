**Underfitting in Machine Learning **
This project explains the concept of Underfitting in Machine Learning in a simple way.

**What I Covered**
1. What is Underfitting
2. Causes of Underfitting
3. Effects (High Bias, Poor Accuracy)
4. Basic implementation

**Key Insight**
Underfitting occurs when a model is too simple to capture patterns in the data.
